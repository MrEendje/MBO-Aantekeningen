## Inleiding

Leren programmeren is voor veel mensen lastig. Computers lezen instructies **extreem letterlijk**, dus als programmeur moet je leren denken in instructies die een computer begrijpt.

## Programmeerregels

- **Order of execution:** code wordt van boven naar beneden uitgevoerd.
    
- **; (semicolon):** geeft het einde van een regel code aan.
    
- **{ } (curly brackets):** definieert een codeblok.

### Types

- **int:** heel getal
    
- **float:** komma-getal
    
- **bool:** true of false
    
- **string:** tekst tussen aanhalingstekens
    

### Dynamic vs Static typed

- **JavaScript:** dynamic typed (let, var, const)
    
- **C#:** static typed (je moet datatype expliciet aangeven)
    

### Voorbeelden

**JavaScript:**

```javascript
let roundedNumber = 2;
let decimalNumber = 3.14;
let isFalse = false;
let playerName = "Shane van der Woude";
```

**C#:**

```csharp
int roundedNumber = 2;
float decimalNumber = 3.14f;
bool isFalse = false;
string playerName = "Shane van der Woude";
```
